Learning Levels

Structured progression from Python basics to machine learning specialization.

🟢 Level 1 — Beginner

15 Hours | 6 Sessions

Build strong programming logic, Python syntax understanding, loops, functions, and mini projects.

Session 1 — Variables & Data Types

Learn int, float, strings, booleans, input/output, and build a student report card program.

Session 2 — Conditionals

Create grade calculators using if, elif, else, comparison operators, and logical operations.

Session 3 — Loops

Master for loops, while loops, patterns, multiplication tables, and countdown timers.

Session 4 — Data Structures

Work with lists, tuples, sets, dictionaries, and build a student marks register.

Session 5 — Functions

Learn reusable functions, parameters, return values, and modular programming.

Session 6 — Mini Project Sprint

Combine all concepts to build complete beginner-level Python applications.

Beginner Mini Projects

  • Quiz Application
  • Stone Paper Scissors Game
  • Number Guessing Game

🔵 Level 2 — Intermediate

18 Hours | 6 Sessions

Build real-world applications using file handling, databases, APIs, object-oriented programming, and GUI development.

Advanced Functions & Modules

Work with lambda, map(), filter(), reduce(), and create custom Python modules.

File Handling

Read/write text and CSV files and build practical data processing systems.

Exception Handling

Create robust production-safe applications using try, except, and finally blocks.

Object-Oriented Programming

Learn classes, objects, inheritance, and professional software design.

MySQL Database Connectivity

Connect Python with databases and perform CRUD operations.

Tkinter GUI & APIs

Create desktop applications and connect to APIs using requests.

Intermediate Mini Projects

  • GUI Calculator Application
  • Dictionary Application
  • Student Record Search System

🔴 Level 3 — Advanced

24 Hours | 8 Sessions

Become industry-ready with data science, machine learning, data visualization, and end-to-end AI workflows.

NumPy

Perform fast numerical computations and large-scale data operations.

Pandas & DataFrames

Load, filter, clean, and analyse real-world datasets professionally.

Data Cleaning

Handle missing values, duplicates, datatype conversion, and preprocessing.

Data Visualization

Create histograms, scatter plots, heatmaps, and business insights visually.

Machine Learning Foundations

Learn Linear Regression, train-test split, MAE, and R² evaluation.

Classification Models

Work with Logistic Regression, Decision Trees, confusion matrices, and accuracy analysis.

Advanced ML Projects

  • Insurance Price Prediction
  • Customer Churn Prediction
  • Laptop Price Prediction
